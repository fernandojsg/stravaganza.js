# This Way

![This Way](thisway.png)

A pixel-accurate JavaScript port of the demoscene production "This Way" by Stravaganza, using Three.js. Built from the original C++ source code.

## Previous versions

- **Original**: [This Way](https://www.pouet.net/prod.php?which=7086) — Win32 demo (Euskal 10, 2002)
- **64kb port**: [This Way 64kb](https://www.pouet.net/prod.php?which=16676)
- **JS port**: [This Way JS](https://www.pouet.net/prod.php?which=62440) — artistic reinterpretation with creative liberties
